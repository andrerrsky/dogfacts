A simple project made with pure HTML, CSS and JS code.

## Made with

1. HTML
2. CSS

4. Dog API - https://kinduff.github.io/dog-api/

# Getting started

1. Run 'npm install'
2. Run 'npm run dev'

If you have "undefined" problems with CORS, I recommend to use this Chrome extension (Allow CORS):
